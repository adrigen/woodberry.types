woodberry.types Installation
----------------------------

To install woodberry.types using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``woodberry.types`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        woodberry.types
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
